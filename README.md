# NewsHunt

## Introduction
NewsHunt is a powerful news aggregation platform that delivers fresh, curated content from various sources. Stay informed with hourly updates and AI-driven recommendations.

## Features
- Hourly updated news  
- Auto-curated content  

## Technologies Used
- Next.js
- React
- Node.js
- Tailwind CSS
- AI-based content curation
